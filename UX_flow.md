---

## First Impression Flow

When user opens site:

1. Floating navbar appears
2. Hero text introduces Vidyush
3. Photo interaction adds personality
4. Background typography reinforces identity

---

## Navigation Map

VIDYUSH → scroll to top

About → scroll to about section

Services → scroll to services

My Work → open work page

LET'S CONNECT → scroll to contact form

---

## Hero Interaction

User actions:

Hover image  
→ slight rotation

Move cursor  
→ image follows cursor slightly (parallax)

Scroll  
→ reveals next section

---

## Services Interaction

User scrolls vertically.

Mouse wheel triggers horizontal movement.

Cards slide left while user scrolls.

Hovering a card:

- card lifts upward
- shadow deepens

---

## Works Interaction

Homepage displays project grid.

Click project card:

→ open live project externally

---

## Contact Interaction

User fills:

Name  
Email  
Message

Click submit.

Form sends message.
