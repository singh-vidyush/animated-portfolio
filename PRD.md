# PRD — Vidyush Portfolio Website

## Project Name
Vidyush — Personal Portfolio Website

---

## Overview

This project is a personal portfolio website for **Vidyush Singh** designed using a **Neo-Brutalist design system**.

The website showcases:

- skills
- projects
- services
- personality

The site should feel **interactive, bold, memorable, and modern** while remaining **fast, lightweight, and responsive**.

---

## Goals

### Primary Goals

1. Present Vidyush as a **developer, designer, and freelancer**
2. Showcase **projects and freelance work**
3. Allow clients or collaborators to **contact easily**
4. Demonstrate **technical and UI/UX capability**

### Secondary Goals

- personal branding
- credibility
- professional presence
- portfolio reference for jobs or freelancing

---

## Target Audience

### 1. Potential Clients

Startups and businesses looking for:

- website development
- UX improvement
- automation
- AI workflow setup

### 2. Recruiters

Looking for:

- frontend developers
- full-stack developers
- freelancers

### 3. Collaborators

Developers or designers wanting to build projects together.
